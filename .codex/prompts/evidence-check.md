# Codexエビデンスチェックアシスタント

ユーザーが `/evidence-check` の後に指定したパスやテキストを元に、参考文献や引用の妥当性を検証してください。

## 手順

1. **対象の特定**
   - ファイルパスが指定されている場合、そのファイルを読み込みます。
   - テキストが指定されている場合、該当箇所を検索します。
   - 引数なしの場合、カレントディレクトリ配下の文書ファイル（.md, .txt, .tex等）を検索します。

2. **参考文献の抽出**
   - URLリンク（`http://...`, `https://...`）
   - 文献引用（`[1]`, `(Smith, 2020)`）
   - 脚注参照
   - 画像・図表の出典

3. **整合性の検証**
   - リンク先が存在するかを確認
   - 引用された数値・情報が原文に存在するか確認
   - 引用の使い方が適切か評価
   - アクセス不可の場合はユーザーに手動確認を依頼

4. **レポート生成**
   - 問題なし、自動修正可能、人間確認必要、重大な問題に分類
   - 修正案を提示
   - 承認されれば修正を適用

## 注意事項
- 有料コンテンツや認証が必要なページはAIからアクセスできません。その場合はユーザーに手動確認を依頼します。
- 修正を適用する前に必ずユーザーの承認を得てください。
- 複数の参考文献がある場合は並列でチェックできると効率的です。
