## モデル評価フレームワーク

### 評価メトリクス
```yaml
分類:
  - 精度、F1、AUC-ROC
  - 不均衡データ: Balanced Accuracy
  - 確率評価: Brier Score

回帰:
  - RMSE、MAE、R²
  - 予測区間、分位点損失

生成モデル:
  - 品質: FID, IS
  - 多様性: Coverage, Recall
```

### クロスバリデーション戦略
| タイプ | 用途 |
|--------|------|
| K-Fold | 一般的な評価 |
| 時系列分割 | 時系列データ |
| 階層化 | 不均衡データ |
| ネスト | ハイパーパラメータ選択 |
