# 批判的思考スキルモジュール

## 批判的思考の実践

### 基本原則

1. **証拠に基づく分析**
   - 情報源の信頼性評価
   - データの妥当性検証
   - 証拠の強度判定

2. **論理的推論**
   - 前提と結論の関係性
   - 論理的誤謬の識別
   - 推論の妥当性確認

3. **バイアスの認識**
   - 認知バイアスの特定
   - 確証バイアスの回避
   - 多角的視点の確保

## 実践的手法

### 1. 議論の分析

**構造分析**
- 主張の特定
- 根拠の抽出
- 推論の評価
- 隠れた前提の発見

**論理的誤謬のチェック**
- 人身攻撃（ad hominem）
- 権威への訴え
- 偽の二分法
- 循環論法
- 一般化の誤り

### 2. 問題解決への応用

**批判的思考プロセス**
1. 問題の定義と範囲設定
2. 仮定の明確化と検証
3. 複数の解決策の生成
4. 各選択肢の批判的評価
5. 実装と結果の検証

### 3. 意思決定支援

**決定分析フレームワーク**
- 目標と制約の明確化
- オプションの網羅的リスト
- 各選択肢の長所短所分析
- リスクと不確実性の評価
- 意思決定基準の明示

## ベストプラクティス

1. **知的謙虚さ**
   - 自分の限界を認識する
   - 間違いを認める準備
   - 学習への開放性

2. **体系的アプローチ**
   - 一貫した分析手法
   - 文書化と追跡可能性
   - 定期的な見直し

3. **協調的批判**
   - 建設的なフィードバック
   - 相手の論点の理解
   - 共通基盤の探索

4. **継続的改善**
   - 思考プロセスの反省
   - 新しい手法の学習
   - 実践からのフィードバック

## よくある落とし穴

1. **確証バイアスの罠**
   - 都合の良い証拠のみ採用
   - 反証の無視
   - 結論ありきの分析

2. **過度の懐疑主義**
   - 実用的判断の遅延
   - 分析の麻痺
   - 相対主義への陥落

3. **感情的反応**
   - 批判と人格攻撃の混同
   - 防御的態度
   - 客観性の喪失

4. **複雑性の過小評価**
   - 単純化の誤り
   - コンテキストの無視
   - システム効果の軽視
