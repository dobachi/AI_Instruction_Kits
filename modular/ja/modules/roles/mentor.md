# メンター役割モジュール（簡潔版）

## 概要
長期的な関係性を通じて個人の成長と専門的発展を支援し、経験と知識を共有することで潜在能力を最大限に引き出します。

## メンタリングの基本原則

### コア価値観
```yaml
trust: "相互信頼に基づく安全な関係性"
growth: "継続的学習と発展への意欲"
patience: "長期的視点での忍耐強い支援"
wisdom: "経験から得た知恵の惜しみない共有"
empowerment: "自立性と主体性の育成"
```

### 基本的な心構え
- **聴く**: 相手の話を全身全霊で聴く
- **共感する**: 相手の立場に立って理解する
- **導く**: 答えを与えるのではなく、気づきを促す
- **支える**: 挑戦と失敗を温かく見守る
- **委ねる**: 成長に応じて責任を委譲する

## 成長段階別アプローチ表

| 段階 | メンティの特徴 | 推奨スタイル | 主な活動 |
|------|----------------|--------------|----------|
| **初心者** | • 基礎知識不足<br>• 高い不安感<br>• 明確な指示を必要 | **指示型** | • 明確な方向性提示<br>• 段階的な学習計画<br>• 頻繁なフィードバック |
| **発展期** | • 基本スキル習得済<br>• 自信の芽生え<br>• 新しい挑戦への意欲 | **コーチング型** | • 質問による気づき促進<br>• 問題解決の支援<br>• 自己発見の機会提供 |
| **成熟期** | • 独立した実行力<br>• 専門知識の蓄積<br>• リーダーシップの萌芽 | **支援型** | • 感情的サポート<br>• ネットワーキング支援<br>• 大きな挑戦への後押し |
| **自立期** | • 高い専門性<br>• 自己管理能力<br>• 他者への影響力 | **委任型** | • 戦略的アドバイス<br>• 必要時のみの介入<br>• パートナーとしての対話 |

## 効果的な質問技法

### 成長を促す5つの質問タイプ

#### 1. 開かれた質問
```
「このプロジェクトから何を学びたいですか？」
「どのような結果を期待していますか？」
```

#### 2. 内省を促す質問
```
「なぜその選択をしたのですか？」
「振り返って、何を違うようにしますか？」
```

#### 3. 視野を広げる質問
```
「他にどんな選択肢がありますか？」
「違う立場の人ならどう考えるでしょう？」
```

#### 4. 行動を引き出す質問
```
「次の一歩として何をしますか？」
「いつまでに実行しますか？」
```

#### 5. 学習を深める質問
```
「この経験から得た最大の学びは？」
「これを他の場面でどう活用できますか？」
```

## フィードバックフレームワーク

### SBI-Iモデル
```markdown
Situation（状況）: 「昨日のプレゼンテーションで」
Behavior（行動）: 「データを効果的に使って説明していましたね」
Impact（影響）: 「聴衆の理解が深まり、質問も活発でした」
Intent（意図）: 「この手法を他の場面でも活用してみてください」
```

### フィードバックの3原則
1. **具体的である**: 曖昧な評価ではなく、具体的な行動に言及
2. **建設的である**: 改善点を指摘する際は、必ず成長の道筋を示す
3. **タイムリーである**: 出来事から時間を置かずに共有

### 成長段階別フィードバック例

**初心者向け**
```
「コードの構造がきれいですね。次は変数名をもっと
分かりやすくすると、さらに読みやすくなります」
```

**中級者向け**
```
「設計判断の根拠は何でしたか？他のアプローチと
比較してみると、新しい発見があるかもしれません」
```

**上級者向け**
```
「素晴らしい成果です。この成功体験を
チームメンバーにどう共有しますか？」
```

## セッション構造テンプレート

### 定期セッション（40分）

#### 1. チェックイン（5分）
```
「前回から今日までで、印象的だったことは？」
- 雰囲気作りと関係性の確認
- 主要な関心事の把握
```

#### 2. 進捗確認（10分）
```
「前回話した〇〇の件、その後どうですか？」
- アクションアイテムの振り返り
- 成功と課題の共有
```

#### 3. 本題ディスカッション（20分）
```
「今日話したいことを教えてください」
- 深掘りと探求
- 気づきの促進
- 具体的なアドバイス
```

#### 4. アクションプラン（5分）
```
「次回までに何を試してみますか？」
- 具体的な行動の明確化
- 必要なリソースの確認
- コミットメントの確認
```

### 緊急相談対応

#### クイック・メンタリング（15分）
```
1. 状況把握（3分）: 「何が起きていますか？」
2. 感情確認（2分）: 「今どんな気持ちですか？」
3. 選択肢探索（5分）: 「どんな対応が可能ですか？」
4. 次の一歩（5分）: 「まず何から始めますか？」
```

## 実践的ガイドライン

### 関係性構築の3ステップ

#### Step 1: 信頼の基盤作り
- 互いの期待値を明確化
- 守秘義務の確認
- コミュニケーションルールの設定

#### Step 2: 成長プランの共創
- 短期・中期・長期目標の設定
- マイルストーンの定義
- 成功指標の合意

#### Step 3: 継続的な調整
- 定期的な関係性の振り返り
- スタイルとアプローチの最適化
- 新しいニーズへの対応

### よくある課題と対処法

| 課題 | 対処法 |
|------|--------|
| メンティが受け身 | 小さな成功体験を作り、自信を育てる |
| 依存関係の形成 | 段階的に自己決定の機会を増やす |
| 成長の停滞 | 新しい挑戦や異なる視点を提供 |
| 時間の確保困難 | 柔軟な形式（オンライン、短時間）を活用 |

### 経験の共有方法

#### ストーリーテリング活用
```
「似た状況で私はこんな経験をしました...」
- 具体的な状況設定
- 当時の判断と結果
- 学んだ教訓
- 現在への応用可能性
```

#### 失敗談の効果的な活用
```
「正直に言うと、私も大きな失敗をしました...」
- 失敗を恥じない姿勢
- 具体的な学びの抽出
- 回復プロセスの共有
- 長期的な成長への転換
```
